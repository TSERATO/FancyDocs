# Commands for the moderation system

<Callout type="warn">
    The command list is not complete and not all commands are properly documented yet.
    Some commands might not be implemented yet.
</Callout>

## Report

### report

description: Reports a player for misconduct.
Syntax: ``/report <player> [reason]``
- `<player>` - Player to report
- `[reason]` - Report reason (optional)

Permission: ``fancycore.commands.report``

### reports

description: Manage player reports.
Syntax: ``/reports list`` or ``/reports view <reportID>`` or ``/reports close <reportID>``
- `list` - Lists all active reports
- `view <reportID>` - View report details (reportID: Report identifier)
- `close <reportID>` - Close a report (reportID: Report identifier)

Permission: ``fancycore.commands.reports.list``, ``fancycore.commands.reports.view``, ``fancycore.commands.reports.close``

## Logs

### modlogs

description: Views the moderation logs.
Syntax: ``/modlogs [player]``
- `[player]` - Filter by player (optional)

Permission: ``fancycore.commands.modlogs``

### chatlogs

description: Views the chat logs.
Syntax: ``/chatlogs [player] [--withCommands]``
- `[player]` - Filter by player (optional)
- `[--withCommands]` - Include commands (optional)

Permission: ``fancycore.commands.chatlogs``
